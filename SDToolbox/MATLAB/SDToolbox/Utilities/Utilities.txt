Shock and Detonation Toolbox 'Utilities' module

Utility tools for creating pre-defined plots and output files from the outputs
of the CJspeed, cvsolve, and zndsolve functions.

This module defines the following functions:

    CJspeed_plot
    cv_plot
    cp_plot
    znd_plot
    znd_fileout

Theory, numerical methods and applications are described in the
following report:

“SDToolbox: Numerical Tools for Shock and Detonation Wave

Browne, J.L. Zeigler, N.P. Bitter, B.E. Schmidt, J. Lawson, J.E.
