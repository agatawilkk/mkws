Shock and Detonation Toolbox 'ZND' module

Calculates ZND structure model of detonations

This module defines the following functions:

    zndsys
    zndsolve

################################################################################
Theory, numerical methods and applications are described in the following report:

“SDToolbox:  Numerical Tools for Shock and Detonation Wave Modeling.”
Explosion Dynamics Laboratory. Contributors: S.T.  Browne, J.L. Zeigler,
N.P. Bitter, B.E. Schmidt, J. Lawson, J.E. Shepherd.
GALCIT Report FM2018.001. California Institute of Technology, Pasadena, CA. 2020.
